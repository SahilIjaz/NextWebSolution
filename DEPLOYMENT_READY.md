## Support

### If Something Breaks
1. Check backend is running: `npm start`
2. Check frontend server running: `python3 -m http.server 5000`
3. Check database connection in logs
4. Check browser console for errors
5. Review documentation files

### Common Tasks
- Restart backend: `pkill -f "node src/server.js" && npm start`
- Clear data: Run any `seed-*.js` file
- Check logs: `tail -f /tmp/backend.log`
- Test API: `curl http://localhost:8000/api/testimonials`
